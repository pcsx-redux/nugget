PSYQo

What?

The PSYQo library is a object oriented C++ library for developing applications that run on the PlayStation 1. It is an opinionated yet lightweight and safe solution for modern development on the PS1.

How?

Please refer to the Getting Started and Concepts pages for more information. While the library makes heavy use of C++, the surface API only requires some basic knowledge of C++. Still, the user of the library is encouraged to get familiar with C++ concepts such as classes, or inheritance.

The library makes uses of the EASTL, and is available to the user as a side effect. However, not all pieces of the library will be guaranteed to work properly at the moment. It has been modified to support the PlayStation1. The goal is still to get most of the EASTL working properly eventually.
