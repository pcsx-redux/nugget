\doxysubsection{Detailed Description}
The \doxylink{structpsyqo_1_1Prim_1_1DrawingAreaStart}{Drawing\+Area\+Start} primitive. 

This primitive will define the start location of the drawing area. It will be used by the {\ttfamily \doxylink{classpsyqo_1_1GPU}{GPU}} class when changing buffers, so there shouldn\textquotesingle{}t be any need to use it directly. Any primitive drawn outside of the drawing area will be clipped away.
