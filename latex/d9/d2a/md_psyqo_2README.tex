 \hypertarget{md_psyqo_2README_autotoc_md60}{}\doxysection{\texorpdfstring{What?}{What?}}\label{md_psyqo_2README_autotoc_md60}
The PSYQo library is a object oriented C++ library for developing applications that run on the Play\+Station 1. It is an opinionated yet lightweight and safe solution for modern development on the PS1.\hypertarget{md_psyqo_2README_autotoc_md61}{}\doxysection{\texorpdfstring{How?}{How?}}\label{md_psyqo_2README_autotoc_md61}
Please refer to the \doxysectlink{md_psyqo_2GETTING__STARTED}{Getting Started}{0} and \doxysectlink{md_psyqo_2CONCEPTS}{Concepts}{0} pages for more information, or you can directly read the \href{https://pcsx-redux.github.io/nugget/d6/d4a/namespacepsyqo.html}{\texttt{ full documentation}}. While the library makes heavy use of C++, the surface API only requires some basic knowledge of C++. Still, the user of the library is encouraged to get familiar with C++ concepts such as \href{https://cplusplus.com/doc/tutorial/classes/}{\texttt{ classes}}, \href{https://en.cppreference.com/w/cpp/language/lambda}{\texttt{ lambdas}}, or \href{https://cplusplus.com/doc/tutorial/inheritance/}{\texttt{ inheritance}}.

The library makes uses of the \href{https://github.com/electronicarts/EASTL}{\texttt{ EASTL}}, and is available to the user as a side effect. However, not all pieces of the library will be guaranteed to work properly at the moment. It has been modified to support the Play\+Station1. The goal is still to get most of the EASTL working properly eventually.\hypertarget{md_psyqo_2README_autotoc_md62}{}\doxysection{\texorpdfstring{Who?}{Who?}}\label{md_psyqo_2README_autotoc_md62}
