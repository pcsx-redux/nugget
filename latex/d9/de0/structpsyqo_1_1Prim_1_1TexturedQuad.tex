\doxysubsection{Detailed Description}
A textured quad. 

This primitive will draw a textured quad. See {\ttfamily \doxylink{structpsyqo_1_1Prim_1_1Quad}{Quad}} for more information about vertices and ordering. The primitive has weird-\/looking ordering of members, but it is necessary to accommodate the way the hardware wants the quad information to be sent to it. The attributes of the primitive can be better visualized with this order\+:
\begin{DoxyItemize}
\item {\ttfamily pointA}, {\ttfamily pointB}, {\ttfamily pointC}, {\ttfamily pointD}
\item {\ttfamily uvA}, {\ttfamily uvB}, {\ttfamily uvC}, {\ttfamily uvD}
\item {\ttfamily clut\+Index}, {\ttfamily tpage} 
\end{DoxyItemize}
