\doxysubsection{Detailed Description}
An asynchronous read request. 

This struct serves as a persistent storage for an asynchronous read request. Its purpose is to be embedded outside of the parser, so it can keep track of the current state of the request. IMPORTANT\+: The buffer pointer needs to hold at least enough sectors to hold the file. Even if the file size isn\textquotesingle{}t a multiple of 2048, the buffer needs to be a multiple of
\begin{DoxyEnumerate}
\item The buffer is not owned by the parser, and it\textquotesingle{}s the user\textquotesingle{}s responsibility to ensure it\textquotesingle{}s valid during the whole request. 
\end{DoxyEnumerate}
