\doxysubsection{Detailed Description}
The \doxylink{structpsyqo_1_1Prim_1_1Sprite}{Sprite} primitive. 

This primitive will draw a sprite of arbitrary size. The {\ttfamily position} member describes the location within the screen where the sprite will be blitted. The {\ttfamily tex\+Info}member indicates where the source sprite is blitted from, and the {\ttfamily size} member specifies actual sprite size to blit.

The texture information needs to be specified with a \doxylink{structpsyqo_1_1Prim_1_1TPage}{TPage} primitive beforehand.
