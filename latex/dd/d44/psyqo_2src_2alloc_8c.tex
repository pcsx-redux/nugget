Returns the pointer to the end of the heap. 

This function will return the pointer to the end of the heap. The heap works lazily, and this function may return a NULL pointer. The heap can grow and shrink depending on usage. Computing the current size of the heap is done by subtracting the start pointer from the end pointer. Note this wouldn\textquotesingle{}t account for memory fragmentation.
