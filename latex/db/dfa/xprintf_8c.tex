Prints a formatted string to a newly allocated string. 

This function is a helper around {\ttfamily vxprintf}, which will print to a string, and otherwise behaves the same as normal glibc (v)asprintf. The string will be allocated using {\ttfamily \doxylink{psyqo_2alloc_8h_a3804a7f46fcc70023cde5a3f007662da}{psyqo\+\_\+malloc()}}, and must be freed using {\ttfamily \doxylink{psyqo_2alloc_8h_a2e8fd619ab9a5b9581d93558dfb4ee76}{psyqo\+\_\+free()}}.
