\subsubsection*{template$<$Concepts\+::\+Is\+CDRom\+Device\+State\+Enum S$>$\newline
class psyqo\+::\+CDRom\+Device\+::\+Action$<$ S $>$}
The action base class for the internal state machine. 

This class is meant to be extended by the various actions that the \doxylink{classpsyqo_1_1CDRom}{CDRom} device can perform. It provides a framework for the state machine that is used to perform various operations. While it is public, it is not meant to be used directly by the application, and should only be used by the \doxylink{classpsyqo_1_1CDRomDevice}{CDRom\+Device} class.
