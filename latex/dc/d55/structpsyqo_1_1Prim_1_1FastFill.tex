\doxysubsection{Detailed Description}
The \doxylink{structpsyqo_1_1Prim_1_1FastFill}{Fast\+Fill} primitive. 

Sending this command will fill the VRAM space specified by the {\ttfamily rect} member with the color specified by the {\ttfamily \doxylink{unionpsyqo_1_1Color}{Color}} argument of the constructor or the {\ttfamily set\+Color} method.

This primitive will ignore the current scissor, so it\textquotesingle{}s not a good idea to use it directly. It\textquotesingle{}ll be used properly by the various \doxylink{classpsyqo_1_1GPU}{GPU} clear commands.
