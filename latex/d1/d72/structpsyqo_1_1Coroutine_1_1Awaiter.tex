\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T = void$>$\newline
struct psyqo\+::\+Coroutine$<$ T $>$\+::\+Awaiter}
The awaiter type. 

The awaiter type is the type that is used to suspend the coroutine after scheduling an asychronous operation. The keyword {\ttfamily co\+\_\+await} can be used on an instance of the object to suspend the current coroutine. Creating an instance of this object is done by calling {\ttfamily coroutine.\+awaiter()}.
