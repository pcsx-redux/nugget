\doxysubsection{Detailed Description}
A task queue for processing tasks sequentially. 

This class is used to process a sequence of tasks one after the other. The design is loosely inspired by the Java\+Script Promise API. A task is effectively a lambda function. A queue can be run multiple times, and can also be embedded within another queue.
