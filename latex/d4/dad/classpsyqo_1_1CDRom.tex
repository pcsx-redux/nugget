\doxysubsection{Detailed Description}
The base \doxylink{classpsyqo_1_1CDRom}{CDRom} class. 

The \doxylink{classpsyqo_1_1CDRom}{CDRom} system is meant to be swappable between multiple implementations, so the base class is a pure abstraction. This allows the ISO9660 parser to be used with any \doxylink{classpsyqo_1_1CDRom}{CDRom} implementation.

Read a sector from the \doxylink{classpsyqo_1_1CDRom}{CDRom}. 

The function will make reasonable attempts at reading the disk, but it is not guaranteed to succeed. Failures may be caused by the disk being faulty, the lid being opened, or no valid disk being present. Only one operation can be in progress at a time.

This is a convenience function that will schedule a read operation and return a task that can be waited on. The difference with {\ttfamily schedule\+Read\+Sectors} is that this method will read its arguments right before the operation is processed, so the request data can be filled in at the last moment.
