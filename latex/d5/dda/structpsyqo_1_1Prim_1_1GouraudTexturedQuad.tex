\doxysubsection{Detailed Description}
A textured, blended quad. 

This primitive will draw a textured quad with its texels blended with the interpolated color values of its vertices. See {\ttfamily \doxylink{structpsyqo_1_1Prim_1_1Quad}{Quad}} for more information about vertices and ordering. The primitive has weird-\/looking ordering of members, but it is necessary to accommodate the way the hardware wants the quad information to be sent to it. The attributes of the primitive can be better visualized with this order\+:
\begin{DoxyItemize}
\item {\ttfamily pointA}, {\ttfamily pointB}, {\ttfamily pointC}, {\ttfamily pointD}
\item {\ttfamily colorA}, {\ttfamily colorB}, {\ttfamily colorC}, {\ttfamily colorD}
\item {\ttfamily uvA}, {\ttfamily uvB}, {\ttfamily uvC}, {\ttfamily uvD}
\item {\ttfamily clut\+Index}, {\ttfamily tpage} Note that {\ttfamily colorA} can only be set using the constructor, or the {\ttfamily set\+ColorA} method. 
\end{DoxyItemize}
