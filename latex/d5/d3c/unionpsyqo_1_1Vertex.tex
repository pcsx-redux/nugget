\doxysubsection{Detailed Description}
The \doxylink{unionpsyqo_1_1Vertex}{Vertex} struct. 

Represents a vertex in the VRAM space. Its binary representation is meant to be the same as the vertices that \doxylink{classpsyqo_1_1GPU}{GPU} commands await. It should fit in a single register, and the MIPS ABI ought to pass it as argument or return value gracefully, making it a nice general purpose utility.

Accessors can be either \{x, y\}, \{u, v\}, \{s, t\}, or \{w, h\}, for readability purposes.

Using C++ list-\/initializations, one can create a vertex directly using such syntax\+:

{\ttfamily \doxylink{unionpsyqo_1_1Vertex}{Vertex}\{\{.x = 18, .y = 42\}\};}
