\doxysubsection{Detailed Description}
The primitive used to begin a polyline. 

This primitive is used to begin a flat-\/color polyline. As polylines are drawn using multiple \doxylink{structpsyqo_1_1Prim_1_1Line}{Line} primitives, this primitive is used to set the color of the polyline and the first vertex. As polylines can be complex, this allows drawing them piece by piece, iteratively. After sending this primitive, you need to send a number of \doxylink{unionpsyqo_1_1Vertex}{Vertex} structures using the \doxylink{classpsyqo_1_1GPU}{GPU}\textquotesingle{}s {\ttfamily send\+Raw} method, and then send a \doxylink{structpsyqo_1_1Prim_1_1PolyLineEnd}{Poly\+Line\+End} struct to finish the polyline. Note that it may be necessary to use the {\ttfamily wait\+Fifo} method of the \doxylink{classpsyqo_1_1GPU}{GPU} to ensure that the \doxylink{classpsyqo_1_1GPU}{GPU} has finished processing the previous vertex data before sending the next one.
