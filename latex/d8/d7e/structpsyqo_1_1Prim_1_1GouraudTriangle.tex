\doxysubsection{Detailed Description}
A gouraud-\/shaded triangle. 

This primitive will draw a gouraud-\/shaded triangle. It will be drawn between the {\ttfamily pointA}, {\ttfamily pointB}, and {\ttfamily pointC}. Its color will be interpolated between the colors of its three vertices. Note that {\ttfamily colorA} can only be set using the constructor, or the {\ttfamily set\+ColorA} method.
