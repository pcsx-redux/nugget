\doxysubsection{Detailed Description}
An ISO9660 parser. 

This class is a simple ISO9660 parser. It is not meant to be a full implementation, but rather a simple parser that can be used to find the files to be loaded.

Read a file asynchronously. 

This method reads a file asynchronously. It will read the file from the given {\ttfamily entry} in the {\ttfamily \doxylink{structpsyqo_1_1ISO9660Parser_1_1ReadRequest}{Read\+Request}}, and will read the number of sectors corresponding to the entry\textquotesingle{}s size. The buffer is specified by the {\ttfamily buffer} field in the {\ttfamily \doxylink{structpsyqo_1_1ISO9660Parser_1_1ReadRequest}{Read\+Request}}, and must be large enough to hold the whole file. This method is mainly a helper around the \doxylink{classpsyqo_1_1CDRom}{CDRom} device\textquotesingle{}s {\ttfamily read\+Sectors} method.
