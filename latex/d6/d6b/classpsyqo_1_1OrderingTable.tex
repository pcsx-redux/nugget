\doxysubsection{Detailed Description}
\subsubsection*{template$<$size\+\_\+t N = 4096$>$\newline
class psyqo\+::\+Ordering\+Table$<$ N $>$}
The ordering table. Used to sort fragments before sending them to the \doxylink{classpsyqo_1_1GPU}{GPU}. 

This class is used to sort fragments before sending them to the \doxylink{classpsyqo_1_1GPU}{GPU}. The \doxylink{classpsyqo_1_1GPU}{GPU} will then process the fragments in the order specified by the ordering table. Since the PS1 \doxylink{classpsyqo_1_1GPU}{GPU} doesn\textquotesingle{}t have any depth buffer, the ordering table is used to roughly sort the fragments by their Z value. Each fragment is assigned a "{}bucket"{} based on its Z value. The fragments inside a bucket are NOT sorted. The {\ttfamily \doxylink{classpsyqo_1_1GPU}{GPU}} class {\ttfamily chain} function can be used to schedule an ordering table to be sent to the \doxylink{classpsyqo_1_1GPU}{GPU}.


\begin{DoxyTemplParams}{Template Parameters}
{\em N} & The number of buckets in the ordering table. The larger the number, the more precise the sorting will be, but the more memory will be used. \\
