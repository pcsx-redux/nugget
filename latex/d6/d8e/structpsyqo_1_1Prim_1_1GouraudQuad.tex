\doxysubsection{Detailed Description}
A gouraud-\/shaded quad. 

This primitive will draw a gouraud-\/shaded quad. See {\ttfamily \doxylink{structpsyqo_1_1Prim_1_1Quad}{Quad}} for more information about vertices and ordering. The color of the quad will be interpolated between the colors of its four vertices. Note that {\ttfamily colorA} can only be set using the constructor, or the {\ttfamily set\+ColorA} method.
