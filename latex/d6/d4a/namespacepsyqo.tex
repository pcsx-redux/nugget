Computes the adler32 checksum of a buffer. 

This is a very fast checksum algorithm, but it is not cryptographically secure. It can be used to detect data corruption. It is possible to chunk large buffers by chaining the checksum of the previous chunk with the checksum of the next using the {\ttfamily sum} parameter. Computes the adler32 checksum of a buffer, optimized for words. 

This is a variant of the {\ttfamily adler32} function, which works exclusively on words. It is faster than the byte-\/oriented version, but the buffer needs to be aligned to a word boundary. It is possible to mix the two versions by using the {\ttfamily sum} parameter as explained in the documentation of the byte-\/oriented version.
