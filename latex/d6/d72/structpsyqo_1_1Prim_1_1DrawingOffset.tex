\doxysubsection{Detailed Description}
The \doxylink{structpsyqo_1_1Prim_1_1DrawingOffset}{Drawing\+Offset} primitive. 

This primitive will define the end location of the drawing area. It will be used by the {\ttfamily \doxylink{classpsyqo_1_1GPU}{GPU}} class when changing buffers, so there shouldn\textquotesingle{}t be any need to use it directly. Any primitive drawn will be shifted by the offset specified in the constructor.
