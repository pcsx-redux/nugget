\doxysubsection{Detailed Description}
A class that reads files from the \doxylink{classpsyqo_1_1CDRom}{CDRom}. 

This class provides a PSYQo path to read files from the \doxylink{classpsyqo_1_1CDRom}{CDRom}. The way to use it is to instantiate the class somewhere persistent, and then call \doxylink{classpsyqo_1_1paths_1_1CDRomLoader_a241a965c6649887bb36211bbbfff2208}{read\+File()} with a callback. The callback will be called with the data of the file, or an empty vector if the file could not be read. This is going to allocate memory in different places. Only one file can be read at a time, but it is safe to call \doxylink{classpsyqo_1_1paths_1_1CDRomLoader_a241a965c6649887bb36211bbbfff2208}{read\+File()} again from the callback.
