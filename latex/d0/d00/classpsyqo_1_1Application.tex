\doxysubsection{Detailed Description}
The application class. 

The application class is the main class of the application. It\textquotesingle{}s supposed to be a singleton object instantiated in your main program. It\textquotesingle{}s responsible for the main loop of the application, and creating the initial scene. It will hold the {\ttfamily \doxylink{classpsyqo_1_1GPU}{GPU}} object that can be used to render primitives throughout the lifetime of the application. The {\ttfamily main} function should simply instantiate an application object and call its {\ttfamily run} method.

{\footnotesize\ttfamily \mbox{\hyperlink{classpsyqo_1_1Scene}{psyqo\+::\+Scene}} \texorpdfstring{$\ast$}{*} psyqo\+::\+Application\+::pop\+Scene (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Pop a scene object from the stack. 

Pops the top scene object from the stack. There can be only one active scene at a time. Popping a scene object will cause the current scene to be teared down and the new top scene, if any, to be started. If the scene stack ends up being empty, the {\ttfamily create\+Scene} method will be called again.

Calling this method when the stack is empty will return {\ttfamily nullptr}.

\begin{DoxyReturn}{Returns}
the popped scene, potentially for deletion if needed.
