\doxysubsection{Detailed Description}
The \doxylink{structpsyqo_1_1Rect}{Rect} struct. 

Represents a rectangle in the VRAM space. Its binary representation is meant to be the same as the rectangles that \doxylink{classpsyqo_1_1GPU}{GPU} commands await. It should fit in two registers, and the MIPS ABI ought to pass it as argument or return value gracefully, making it a nice general purpose utility.

Accessors can be either \{a, b\}, or \{pos, size\}, for readability purposes.

Using C++ list-\/initializations, one can create a rectangle directly using such syntax\+:

{\ttfamily \doxylink{structpsyqo_1_1Rect}{Rect}\{.a = \{.x = 18, .y = 42\}, .b = \{\}\};}
