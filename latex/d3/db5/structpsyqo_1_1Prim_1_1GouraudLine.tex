\doxysubsection{Detailed Description}
The Gouraud-\/shaded \doxylink{structpsyqo_1_1Prim_1_1Line}{Line} primitive. 

The \doxylink{structpsyqo_1_1Prim_1_1Line}{Line} primitive will draw a single, gouraud-\/shaded segment. It will be drawn between the {\ttfamily pointA} and {\ttfamily pointB} vertices. The color of the segment will be interpolated between the {\ttfamily colorA} and {\ttfamily colorB} colors. Note that {\ttfamily colorA} can only be set using the constructor, or the {\ttfamily set\+ColorA} method.
