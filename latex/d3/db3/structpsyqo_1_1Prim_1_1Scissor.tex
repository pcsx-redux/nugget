\doxysubsection{Detailed Description}
A compounded \doxylink{structpsyqo_1_1Prim_1_1Scissor}{Scissor} primitive. 

This compounds the necessary primitives to alter the drawing area for all drawing commands. It\textquotesingle{}s meant to be used by the {\ttfamily \doxylink{classpsyqo_1_1GPU}{GPU}} class, but can be used sometimes to clip a different area within the vram.

The default constructor will create a \doxylink{structpsyqo_1_1Prim_1_1Scissor}{Scissor} which will clip over the entire VRAM, effectively disabling scissoring.
