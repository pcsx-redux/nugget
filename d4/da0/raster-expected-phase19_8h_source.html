29// Phase-19 expected hardware-truth values for the affine UV
30// stride-sign + cross-axis sweep.
31//
32// Six triangles, identical 10x10 screen footprint to phase-18, varying
33// in stride sign and per-axis stride direction:
34//
35// T_NEG_U_K05 dU/dx = -0.5 (mirrored U, mild)
36// T_NEG_V_K05 dV/dy = -0.5 (mirrored V, mild)
37// T_NEG_BOTH_K05 dU/dx = -0.5, dV/dy = -0.5
38// T_NEG_U_K16 dU/dx = -1.6 (mirrored U, stretched)
39// T_CROSS_45_K05 45-deg UV rotation (dU/dx = dU/dy = +/-0.5)
40// T_CROSS_90_K16 90-deg UV rotation, stretched
41// (dU/dy = dV/dx = 1.6)
42//
43// Probes are the same five screen positions as phase-18:
44// P_VERTEX (5, 5)
45// P_TOP_NEAR (6, 5)
46// P_LEFT_NEAR (5, 6)
47// P_INTERIOR (8, 8)
48// P_TOP_FAR (12, 5)
49//
50// Predictions seeded under phase-17/18 model:
51// per-axis u_sampled = floor(u_real(x, y) + 0.5)
52// per-axis v_sampled = floor(v_real(x, y) + 0.5)
53// with u_real and v_real evaluated from the affine map. The model
54// is direction-symmetric (works for negative stride) and per-axis
55// independent (works for cross-axis stride). Phase-19 tests both
56// assumptions against silicon.
57//
58// Texture: phase-17 TEX17. Probe decoding:
59// vram(u, v) = u | (v << 5) | (((u + v) & 31) | 1) << 10
60//
61// Status markers:
62// HW_VERIFIED confirmed on SCPH-5501 hardware.
63// HW_TODO sentinel; this probe is uncaptured.
64
