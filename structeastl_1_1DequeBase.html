Detailed Description

template<typename T, typename Allocator, unsigned kDequeSubarraySize>
struct eastl::DequeBase< T, Allocator, kDequeSubarraySize >

DequeBase

The DequeBase implements memory allocation for deque. See VectorBase (class vector) for an explanation of why we create this separate base class.
