Detailed Description

template<typename Value, bool bConst>
struct eastl::intrusive_hashtable_iterator< Value, bConst >

intrusive_hashtable_iterator

An intrusive_hashtable_iterator iterates the entire hash table and not just nodes within a single bucket. Users in general will use a hash table iterator much more often, as it is much like other container iterators (e.g. vector::iterator).

The bConst parameter defines if the iterator is a const_iterator or an iterator.
