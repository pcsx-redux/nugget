template<typename Value>
struct eastl::intrusive_hashtable_iterator_base< Value >

intrusive_hashtable_iterator_base

An intrusive_hashtable_iterator_base iterates the entire hash table and not just nodes within a single bucket. Users in general will use a hash table iterator much more often, as it is much like other container iterators (e.g. vector::iterator).

We define a base class here because it is shared by both const and non-const iterators.
