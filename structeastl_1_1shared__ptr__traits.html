Detailed Description

template<typename T>
struct eastl::shared_ptr_traits< T >

shared_ptr_traits This exists for the sole purpose of creating a typedef called reference_type which is specialized for type void. The reason for this is that shared_ptr::operator*() returns a reference to T but if T is void, it needs to return void, not *void, as the latter is not valid C++.
