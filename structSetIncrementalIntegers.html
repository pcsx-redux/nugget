template<typename T>
struct SetIncrementalIntegers< T >

SetIncrementalIntegers

Used to seed containers with incremental values based on integers.

Example usage: vector<int> v(10, 0); for_each(v.begin(), v.end(), SetIncrementalIntegers<int>()); // v will now have 0, 1, 2, ... 8, 9.
