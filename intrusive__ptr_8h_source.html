21  // We provide default implementations of AddRef and Release in the eastl namespace.
22  // The user can override these on a per-class basis by defining their own specialized
23  // intrusive_ptr_add_ref and intrusive_ptr_release functions. User-defined specializations
24  // do not need to exist in the eastl namespace, but should preferably be in the namespace
