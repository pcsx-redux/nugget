consteval uint32_t psyqo::timer_literals::operator""_ns ( unsigned long long int  value)

Literal operators for time units.

These operators can be used to specify time units suitable for the GPU's armTimer and armPeriodicTimer methods. For example, gpu().armPeriodicTimer(1_s, callback) will create a timer that fires every second.
