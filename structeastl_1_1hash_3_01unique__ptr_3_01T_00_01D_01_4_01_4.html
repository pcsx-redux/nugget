Detailed Description

template<typename T, typename D>
struct eastl::hash< unique_ptr< T, D > >

hash specialization for unique_ptr. It simply returns eastl::hash(x.get()). If your unique_ptr pointer type (the return value of unique_ptr<T>::get) is a custom type and not a built-in pointer type then you will need to independently define eastl::hash for that type.
